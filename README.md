# Plan de Recibo

Dashboard Streamlit para importar el archivo PLAN DE RECIBO y generar los modulos de control:

- Indicadores
- Proveedores y causales
- Recibo y materiales
- Conciliacion
- Base

## Ejecutar local

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Streamlit Cloud

Usa `streamlit_app.py` como archivo principal de la app.
